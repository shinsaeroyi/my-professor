# 🎓 Professor Agent

AI chatbot that mimics a professor's research philosophy, speaking style, and feedback patterns based on their published papers.

## Features

- **Paper auto-collection** — Search by name, auto-fetch titles, abstracts, and citation counts
- **Speaking style presets** — Choose from 5 personality types
- **Photo upload** — Set a real profile photo
- **Research philosophy** — Input what they emphasize, dislike, and frequently say
- **Feedback learning** — Paste real comments to improve accuracy

## Quick Start

### Prerequisites
- Python 3.8+
- Gemini API key (free) from [Google AI Studio](https://aistudio.google.com/)

### Run
1. Download or clone this repo
2. Double-click `start.bat` (auto-installs dependencies on first run)
3. Enter your Gemini API key in Settings (bottom-left gear icon)
4. Click `+ Add Professor` and start chatting

### Stop
Close the terminal window.

## Tech Stack

- Flask / Google Gemini API (free) / OpenAlex API (free) / Vanilla JS
